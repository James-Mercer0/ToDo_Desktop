**To Do Desktop - A simple Java based todo list app for your desktop**


This is a personal project of mine, a simple Java applet that allows you to create a to-do list, keep track of, prioritize, and re-order your tasks.

This is currently a work-in-progress, but has several features:
 - A simple adjustable window - size and location of the window is retained on restarting the app.
 - You can add new tasks, set a priority number and task information for them, and delete tasks you've completed.
 - Tasks can be re-ordered in your list individually or collectively based on the priority you've set.
 - Tasks can be updated as needed.

The main purpose of this project is for me to practice my coding skills and to create an application that I, myself, would enjoy using to track and prioritise tasks in my own daily life. As such, I've been enjoying creating an application with features that I personally will enjoy, in addition to fielding ideas from others to implement in the project.
