**To Do Desktop - A simple todo list app for your desktop**


This is a personal project of mine, a simple Java applet that allows you to create a to-do list and keep track of your tasks.

This is currently a work-in-progress, but has several features:
A simple adjustable window.
You can add tasks, set a priority number, and set a description for each task.
Delete tasks you've completed.
Edit tasks as needed.

As mentioned, this is a WIP and current plans are to add a number of features:
  Re-ordering based on priority
  Keeping track of size/location of the window
  Additional colour themes
  Moving tasks up/down the list

Thank you for taking a look at my project!
